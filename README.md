# NOXKIT

NOXKIT is a Python-based security toolkit designed to provide simple and practical security utilities through a graphical user interface.

---

## Project Status

Planning Phase

---

## Planned Features

- Password Analyzer
- Hash Generator
- File Integrity Checker

---

## Technologies

- Python
- Streamlit

---

## Project Structure

The project will follow a modular architecture to keep each security tool independent and reusable.
